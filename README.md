### Desktop
Create desktop apps in Javascript. Lightweight alternative to Electron because of only supporting macOS (for now).

*Features*
 - [x] Tiny App sizes (macOS: < 100K, Windows: N/A)
 - [x] Debuggable
 - [ ] Fullscreen API
 - [ ] File Read/Write
 - [ ] Menu
